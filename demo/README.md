# Demo: Fix the Markdown Link Checker

A small Python CLI tool that checks markdown files for broken links.
It has 3 bugs. Ask Pi to find and fix them.

## The Task

Run Pi in this directory and prompt it:

```
The markdown link checker has bugs. Run the tests, find the failures, and fix the code.
```

## Expected Outcome

Pi should:
1. Read the code and tests
2. Run the tests (3 failures)
3. Diagnose and fix each bug
4. Re-run tests until all pass
